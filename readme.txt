== Description ==

If you answer comments, you can get information through the blog address of comments.
The other way, you can see information about comments other blogs send by admin e-mail.

댓글에 답글을 달면 댓글의 블로그 주소로 답글의 정보를 보냅니다.
반대로 다른 블로그에서 보낸 답글의 정보는 admin 메일로 받아 볼 수 있습니다.


== Installation ==

1. Upload the full directory into your 'wp-content/plugins' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

1. 전체 디렉터리를 'wp-content/plugins'에 넣습니다.
2. 워드프레스의 'Plugins' 메뉴에서 플러그인을 활성화 시킵니다.
